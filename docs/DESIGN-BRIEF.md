# БТ для дизайна — BookmarkBrain Mini App

Бриф для отрисовки всех текущих экранов в финальном (продакшн) дизайне.
Описывает **то, что уже реализовано** в коде — нужно довести визуал до «докрученного»
качества, сохранив единый язык ДС. После дизайна экраны вернутся на доработку в код.

---

## 1. Продукт и платформа

**BookmarkBrain** — «второй мозг» в Telegram. Пользователь шлёт боту мысли/ссылки/голос,
AI их классифицирует, тегирует, строит списки задач и напоминания. **Mini App** —
визуальный клиент поверх этого: лента сохранённого, поиск по смыслу, пространства
(папки), профиль, и редактирование заметок/списков.

- **Платформа:** Telegram Mini App (WebView внутри Telegram, iOS + Android). Ширина ~360–420px, контент во весь экран, есть `safe-area` сверху/снизу.
- **Стек (для контекста):** React + TypeScript, инлайн-стили на CSS-переменных ДС.
- **Навигация:** не многостраничный сайт, а **один кадр + bottom-sheets** (state-driven). Это влияет на дизайн: переходы между табами — кросс-фейд, не «страницы»; всё модальное — нижние шторки.

## 2. Дизайн-система — жёсткие ограничения

Источник истины токенов: `src/styles/tokens.css`, активная тема **`echo` (light)**.

**Тема залочена:** только светлая, «бумажно-тёплая» (Claude-paper). **Тёмную тему не вводить.** Telegram-тему не пробрасываем (бренд не должен синеть от Telegram).

**Палитра (echo, sage):**

| `--brand-primary-press` | `#547654` | нажатие/насыщенный акцент |
| `--brand-primary-tint` | `#E2EDE2` | мягкая подложка акцента |

**Шрифты:**
- `--font-ui` — **Onest** (основной UI, sans).

**Радиусы:** chips 6 · inputs/small 10 · cards 16 · крупные карточки/тайлы 22 · шторки/hero 28 · pill 999.

**Иконки:** slim-line, stroke 1.5–1.8, `currentColor`, 24×24 viewbox.

**Анимации:** `--ease-out: cubic-bezier(0.2,0.8,0.2,1)`; шторки — выезд снизу (`sheetUp` 320мс); тосты — fade (`toastIn`); смена экрана — кросс-фейд.

## 3. Глобальная навигация

- **BottomNav** — плавающая «таблетка» (frosted) из **3 равных ячеек**: `мысли` · `пространства` · `я`. Иконка + всегда видимый лейбл. Активная вкладка — sage-tint капсула вокруг иконки+лейбла. Рядом отдельная круглая **FAB** (sage, `+`) — создать заметку.
- Позиционирование: `position: fixed`, отступ снизу + `safe-area`. Контент скроллится под неё (у экранов нижний паддинг ~116px).
- **Telegram BackButton** (системная кнопка «назад») закрывает: шторку → деталь → поиск (по приоритету).

## 4. Экраны (5)

### 4.1. Мысли (главная лента) — `мысли ·`
Лента всего сохранённого, как чат-история.
- **Шапка:** крупный заголовок `мысли` + sage-точка (Lora-курсив); справа — пилюля-«колокольчик» с числом активных напоминаний + общий счётчик закладок (mono).
- **Поиск-бар** (pill, frosted) — тап открывает экран Поиска.
- **Переключатель вида** `chat / cards` (сегмент) + метка периода справа (`сегодня`, mono-caps).
- **Фильтр-чипы:** `все · ★ · задачи · голос` (горизонтальный скролл).
- **(Опционально) AI-подсказки** — пейджер карточек («3 закладки про X — собрать в пространство?», «год назад ты сохранил Y — открыть?»). Сейчас скрыты за флагом до реального источника — **нужен дизайн состояния**.
- **Лента (chat-вид):** разделители дней (`сегодня` / `вчера` / `16.05 пт` / `16.05`), строки `ChatRow`: аватар-глиф (цветной кружок по типу), заголовок, время (mono), превью-строка, иконка `⋯` (действия), значки (★ избранное, ✓ выполненный список, прогресс `1/3`).
- **Лента (cards-вид):** карточки `BookmarkCard` (крупнее, с тегами/прогрессом).
- **Состояния:** загрузка (пульсирующая точка), пусто («пока пусто» / «ничего по фильтру»).

### 4.2. Поиск — `поиск`
- Шапка: круглая кнопка «назад» + заголовок.
- Активное поле поиска (pill с sage-обводкой и свечением).
- **Динамический поиск** (debounce 350мс, без Enter).
- **AI-ответ по сохранённому** — выделенный sage-блок с `✦` и курсивным (Lora) ответом-саммари.
- Результаты — карточки (источник-домен, время, заголовок, курсивное саммари).
- Состояния: загрузка, стартовое («о чём подумать?» + примеры), пусто («ничего не нашлось»).

### 4.3. Пространства (папки) — `пространства ·`
- Шапка + курсивная подпись («N пространств · AI собирает похожее само»).
- **Грид 2 колонки** из тайлов-пространств: цветной глиф-бейдж (градиент по тону), название, счётчик закладок (mono).
- Тайл **«создать»** (пунктирная граница, `+`, подпись «или дай AI собрать»).
- ⚠️ Сейчас открытие пространства и создание — заглушки. **Нужен дизайн:** экран/шит просмотра пространства (список закладок папки) и флоу создания.

### 4.4. Я (профиль) — `я ·`
- Шапка + карточка профиля: круглый аватар-инициал (градиент), `@handle`, «N закладок · с DD мес».
- **3 стата** (мини-карточки): всего / на неделе / прочитал (числа — Lora-курсив, sage).
- **Группы настроек** (стеклянные списки строк «лейбл → значение»):
  - Приватность: тихий режим, AI-подсказки, таймзона.
  - Вид: тема, архивировать через, плотность списка.
  - Данные: экспорт в Markdown →, подключить Notion →.
- Подпись-версия внизу (Lora-курсив).
- ⚠️ Часть значений — плейсхолдеры/«в разработке». **Нужен дизайн** реальных контролов (тогглы, селекты).

### 4.5. Заметка (detail) — `заметка`
Открывается тапом по строке/карточке.
- Шапка: «назад» + заголовок + `⋯` (действия).
- Карточка заметки: мета-строка (домен · время · «ai обрабатывает…»), заголовок, курсивное (Lora) саммари.
- **Редактор списка задач** (если заметка — task_list) — см. §6.
- Полный текст (`raw_text`), кнопка «открыть источник» (если есть ссылка), теги (чипы), блок **«ключевые мысли»** (`✦` + список).

## 5. Оверлеи (bottom sheets) и нотификации

Все — нижние шторки (frosted, радиус 28, «ручка» сверху). У всех **× для закрытия** + закрытие тапом по затемнению. Фон под шторкой не скроллится.

1. **ActionSheet** (по `⋯` на заметке/строке): действия над закладкой — напомнить, ★ избранное, переместить в пространство, удалить. Шапка показывает контекст (тип/заголовок).
2. **RemindersSheet** (по колокольчику): список напоминаний, сгруппированы `сегодня / завтра / на неделе / позже`. Строка: аватар, текст задачи/заметки, время (mono), кнопки «отложить» (часы) и «×». Удаление → **undo-тост** «напоминание удалено · отменить» (4с). Пусто: «напоминаний пока нет».
3. **ReminderPickerSheet** («напомнить»): контекст-чип `«…»`; радио-пресеты (сегодня вечером / завтра утром / на выходные / через неделю) + опция **«выбрать дату и время»** → раскрывается **DS-календарь** + **колёса времени** (часы:минуты, iOS-style scroll со snap, центральная sage-подсветка). Кнопка «напомнить».
4. **DatePickerSheet** (дедлайн пункта списка): контекст-чип, **DS-календарь** (неделя с Пн, навигация по месяцам, выбранный день sage-заливкой, сегодня — sage-обводкой), кнопки «снять» / «готово».
5. **MoveToSpaceSheet**: список пространств (бейдж + название + счётчик) + «создать».
6. **QuickCreateSheet** (по FAB): поле быстрой заметки + «сохранить». ⚠️ Сейчас нет загрузки голоса/файлов — **в планах**, можно заложить дизайн вложений.
7. **Kebab-меню пункта списка** (по `⋮` в строке задачи): копировать · напомнить/изменить дедлайн · удалить (красным).
8. **Тосты:** инфо-тост сверху по центру (тёмная пилюля, «в разработке» / «скопировано» / «не сохранилось»); undo-snackbar для удаления (пункта списка — снизу; напоминания — сверху).

## 6. Ключевой паттерн: редактор списка задач (в Заметке)

Интерактивный чек-лист. Строка пункта:
- **Чекбокс** (sage-заливка + галочка когда выполнено; тап тогглит, зона тапа ~28px). Выполненный пункт — зачёркнут, тускнеет.
- **Текст** — тап превращает в инлайн-инпут (белый фон, hairline, видимое sage-выделение); Enter/blur сохраняет, Esc отменяет.
- **Чип дедлайна** (если есть) — `📅 16.05`, тап → календарь, `×` снять.
- **Действия** (на ховере/фокусе, прибиты к низу): `×` (очистить текст) + `⋮` (kebab-меню).
- **Удаление** пункта (через очистку или меню) → **undo-snackbar** 4с.
- **Строка «+ добавить пункт»** внизу: тап → инпут, Enter добавляет и держит фокус.
- Прогресс списка отражается в карточке ленты (`1/3`, done-tick).

## 7. Что нужно от дизайна

1. **Довести все 5 экранов + оверлеи до продакшн-качества** в едином языке ДС (sage echo, стекло, Onest/Lora/Mono).
2. Проработать **все состояния**: загрузка, пусто, ошибка, длинный контент, переполнение.
3. **Дозаложить недостающее:** просмотр пространства (список закладок) + создание; реальные контролы настроек в «Я»; состояние AI-подсказок; вложения (голос/файл) в QuickCreate.
4. **Микро-детали:** иконки, отступы, тени, типографика, активные/нажатые состояния, фокус.
5. Учитывать **узкий вьюпорт** Telegram (360–420px) и `safe-area`.

## 8. Ограничения (не нарушать)

- Только **светлая** тема echo; бренд **sage `#7A9C7A`** — не менять на синий/иной.
- Эстетика «жидкого стекла» (frosted) — сохранить.
- Навигация — табы + нижние шторки (без многостраничности).
- Шрифты строго Onest / Lora / JetBrains Mono.
- Иконки slim-line, одно семейство.
